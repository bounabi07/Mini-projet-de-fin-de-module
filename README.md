# 📖 Livre d'Or — Mini Projet Web

## Description du projet
Notre site est un Livre d'Or électronique permettant aux visiteurs 
de laisser leurs commentaires, avis et messages. Il comprend une 
interface simple et élégante pour consulter les messages existants 
et en ajouter de nouveaux, tout en offrant une expérience utilisateur 
fluide et agréable. Le site sera développé en utilisant HTML, CSS et JavaScript.
